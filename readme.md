# 📊 Power BI Sales Dashboard Project

## 📌 Project Overview
This project presents an interactive Sales Dashboard built using Power BI to analyze and visualize retail sales data. The dashboard provides meaningful insights into sales performance across regions, categories, and customer segments.

---

## 🎯 Objective
- Analyze sales data to identify trends and patterns  
- Track key performance indicators (KPIs)  
- Enable interactive exploration of data using filters and visuals  
- Support data-driven business decisions  

---

## 📂 Dataset Description
The dataset includes the following fields:

- Order ID  
- Order Date & Ship Date  
- Customer Name & Segment  
- Region, State, City  
- Product Category & Sub-Category  
- Sales Amount  

---

## 🛠 Tools & Technologies Used
- Power BI Desktop  
- Power Query (Data Cleaning & Transformation)  
- DAX (Data Analysis Expressions)  
- Data Modeling  
- Data Visualization  

---

## 📊 Dashboard Features

### 🔹 KPI Cards
- Total Sales  
- Average Sales  
- Total Orders  
- Minimum & Maximum Sales  

### 🔹 Interactive Filters (Slicers)
- Region  
- State  
- Category  
- Sub-Category  
- Segment  
- Date  

### 🔹 Visualizations
- Sales Trend Over Time (Line Chart)  
- Sales by Category & Sub-Category (Bar Chart)  
- Regional Sales Distribution (Map Visual)  
- Segment-wise Sales Comparison (Donut/Pie Chart)  
- Ship Mode Analysis  
- Top Performing Products  

---

## 📈 Key Insights
- The **West region** contributes the highest share of sales  
- **Technology category** shows strong performance  
- Seasonal trends indicate peak sales in specific months  
- Consumer segment generates the most revenue  
- Certain products consistently outperform others
